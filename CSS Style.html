Selectors, Declaration Blocks, Properties and Values

CSS starts with a SELECTOR, then a DECLARATION BLOCK that contains the PROPERTY
and VALUE

EX: p (selector) { width: (property) 50%; (value) }

EX: li (selector { color: (property) blue; (value) }

DECLARATION BLOCKS: ^^^^^ { width: 50%;} and {color: blue;}

CSS TYPE Selectors

Tells the browser which rule the element will effect.

EX: p {color:blue;}, 'p' is the Selector

EX: em {font-size: large;} 'em' is the selector

You can even use: body {font-family: cursive;}, which would make this style apply to the whole body
of the document

Type Selectors can be very useful if you need to apply styling to a wide rang of the body of the document

EX: To apply styling to headers (h1) : h1 {font-size: 48pt; font-family: sans-serif; }

EX: h2 header (h2): h2 {font-sze: 32pt; font-family: sans-serif;}

EX: paragraph (p): p {font-family:fantasy;}

Selectors CLASS and ID

Class and ID can be useful when your page needs different attributes, like one in the main article and
others in the sidebar. Add the 'class' element in HTML and then use CSS with the same name as the 'class'
you specified. To use the 'class' in CSS, use a '.' (period) as the selector before the class name,
like so: introduction would be .introduction
and followup would be .followup

HTML

EX: < p class="introduction"> This is a paragraph with a class. < /p>

EX: < p class="followup"> This has a different class. < /p>

CSS

EX: .introduction {font-size: larger;}

EX: .followup {color: dimgray;}

  • Use a CLASS when there could be several elements that need the same style

  • Use an ID when there's only one element to apply it to

HTML

EX: < div id="main-article">CSS makes puppies giggle! < /div> EX: < div id="sidebar" Okay, just kidding. < /div>

CSS

Add the '#' in CSS as the selector for ID'

EX: #sidebar {float: right; width: 20%; border: 1px solid green;}

EX: #main-article {property: value; property: value}
